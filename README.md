# PhotoStream

A photo library built with Angular 18 and Angular Material. Browse an infinite random photostream, save photos to favorites, and view them in detail.

## Prerequisites

- Node.js (v18+)
- Angular CLI (`npm install -g @angular/cli`)

## Setup

```bash
npm install
```

## Run

```bash
ng serve
```

Open [http://localhost:4200](http://localhost:4200)

## Test

```bash
ng test
```

## Build

```bash
ng build
```

Output goes to `dist/xm-project/`.
